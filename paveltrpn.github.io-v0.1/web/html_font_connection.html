ПОДКЛЮЧЕНИЕ ШРИФТОВ В HTML

Подключение шрифта локально в CSS 

    @font-face{
        font-family: 'MyWebFont';
        src: url('../fonts/WebFont.eot');
        src: url('../fonts/WebFont.eot?iefix') format('eot'),
        url('../fonts/WebFont.woff') format('woff'),
        url('../fonts/WebFont.ttf') format('truetype'),
        url('../fonts/WebFont.svg#webfont') format('svg');
        font-weight: normal;
        font-style: normal;
        }

или, может быть, лучше так (как советует Mozilla): 

    @font-face{
        font-family: MyHelvetica;
        src: local("Helvetica Neue Bold"),
             local("HelveticaNeue-Bold"),
             url(MgOpenModernaBold.ttf);
        font-weight: bold;
        }

Подключение шрифта через Gogle Fonts

Первый этап установки google fonts: заходим на веб-страницу google.com/fonts и выбираем понравившийся шрифт. Для более детального подбора можно воспользоваться вспомогательными фильтрами и инструментами.

Вставляем сгенерированный кусок кода, например: 

    <head> 
    <link href='http://fonts.googleapis.com/css?family=Open+Sans:400italic,400&subset=latin,cyrillic' 
        rel='stylesheet' type='text/css'>)
    </head>

или в CSS: 

    @import url(http://fonts.googleapis.com/css?family=Open+Sans:400italic,400&subset=latin,cyrillic)
