Тип Array

Тип Array (массив) в typescript это упорядоченный список элементов одного типа. Что-бы объявить массив, содержащий элементы определённого типа можно использовать следующее объявление: 

    let arrayName: type[];

Например вот так определяется массив из строк: 

    let skills: string[];

Можно добавить один или несколько элементов-строк в массив: 

    skills[0] = "Problem Solving";
    skills[1] = "Programming";

С использованием метода push(): 

    skills.push('Software Design');

Эта строка объявляет массив строк и присваивает ему значения: 

    let skills = ['Problem Sovling','Software Design','Programming'];
    // Или вот так
    let skills: string[];
    skills = ['Problem Sovling','Software Design','Programming'];

Можно сделать "type inference" (вывод типа): 

    let skill = skills[0];
    console.log(typeof(skill));
    // выведет - string
